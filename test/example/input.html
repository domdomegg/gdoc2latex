gdoc2latex

Converts Google Docs files to LaTeX

Usage

Here’s some content, originally written in Google Docs. It demonstrates using some of the features of gdoc2latex. It was downloaded with File > Download > Web page (.html).

Supported features

Things like bold and italics among other things are supported.

Other features include:

  • Lists
  • Underline
  • References (use BibTeX in a Google Docs footnote)[1]
  • Footnotes[2]
  • Both superscript and subscript
  • Links to the web, to an email address or within a document
